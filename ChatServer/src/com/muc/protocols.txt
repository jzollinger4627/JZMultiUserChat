User & Server

1. User --> Server

    * login / logoff
    * status

2. Server --> User

    * online / offline

3. User --> User

    * direct messages
    * broadcast messages / group messaging

Commands:

    login <user> <password>
    logoff

    msg <user> text....
    guest "msg jacob Hello World" <-- sent
    jim: "msg guest Hello World" <--recv

    #topic <-- chatroom / group chat
    join #topic
    leave #topic
    msg #topic body...
